# MySQLSchoolProjects
Ce répertoire contient 2 des travaux de laboratoire que j'ai eu à faire dans le cadre d'un cours sur les bases de données relationnelles. 

Le TP4 consiste à interagir avec une base de données à travers un programme en java JDBC, qui communique avec des requêtes MySQL. Il fallait implanter des triggers sur une base de données, créée par notre binôme auparavant, pour ensuite traduire en requêtes SQL des commandes et les exécuter à travers un programme en Java (JDBC). 

Le TP5 nous a fait travailler avec un système de gestion de base de données NoSQL (MongoDB). Il fallait créer préalablement une base de données sur MongoDB et la remplir avec des données fournies. Par la suite on devait traduire en requêtes SQL des commandes et les présenter à travers un programme en Java (JDBC).
